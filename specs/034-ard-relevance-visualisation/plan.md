# Implementation Plan: ARD Relevance Visualisation

**Branch**: `034-ard-relevance-visualisation` | **Date**: 2025-07-18 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `/specs/034-ard-relevance-visualisation/spec.md`

## Summary

Add ARD (Automatic Relevance Determination) lengthscale extraction and visualisation to all 8 week 11 review notebooks (F1–F8). Each notebook gets new cells that fit a SingleTaskGP on the existing data, extract per-dimension lengthscales, print a raw lengthscale table, compute normalised relevance scores, and display a horizontal bar chart. F7 (neural network surrogate) requires a separate diagnostic GP since its production surrogate has no kernel.
