As such, code should be as simple as possible with each step cleary explained.  
The code must be submitted in the form of Jupyter notebooks.  
The project is a black box optimisation challenge for which there are 8 separate black box optimisation problems.  
Each problem should be solved in it's won jupyter notebook, stored in it's own folder withing the ./functions project folder.  
The folders and ipynb files already exist.  
The data for each problem is stored in the ./data folder structure, with a individual folder for each problem.
The initial data provided with the problem is stored in files named initial_inputs.npy and initial_outputs.npy
Use the Botorch library.
No unit tests are required
